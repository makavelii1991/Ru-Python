# FileManager  Бот для телеграма @brozeph_bot. 
-----------------------------------------------------------------------------------------
1. Скачать всю директорию
2. Исправить в FileManBot.py строку adr. Заменить путь до папки cache Вашим.
3. Запустить FileManBot.py IDLE.
4. В телеграм находим @brozeph_bot.
5. Создаем, удаляем, перезаписываем папки и файлы в меню бота. 
Все файлы сохраняются в папку cache на вашем пк.

------------------------------------------------------------------------------------------
Написан код, выполняющий на стороне сервера следующие опреации:
1. Выдать список файлов.
2. Сохранить файл с содержимым.
3. Выдать содержимое файла.
4. Выдать информацию о файле.
5. Удалить файл с содержимым.
4. Создать папку.
5. Удалить папку.
