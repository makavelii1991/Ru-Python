Условие задачи:
Написать удалённый менеджер файлов (сервер + протокол обмена). 
Должны поддерживаться следующие запросы:
Выдать список файлов.
Сохранить файл с содержимым.
Выдать содержимое файла.
Выдать информацию о файле.
Удалить файл с содержимым.
Создать папку.
Удалить папку.

----------------------------------------------------------------
Порядок запуска программы:
1. Нам нужны: FileManServer.py и ncat.zip
2. Размещаем ncat на С:/, для удобства.
3. Открываем в Блокноте FileManServer.py, меняем путь у переменной adr.
adr - переменная содержащая в себе конечную папку, в которую будут сохранятся и удаляться файлы.
4. Запускаем в терминале FileManServer.py.
5. Запускаем в другом терминале ncat и прописываем команду: ncat localhost 3333
6. Все, готово.
